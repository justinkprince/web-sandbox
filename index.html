Git with audio

--with-audio is an undocumented feature of git-graph. The results are sometimes very strange and hilarious.

To hear an example, just run the following within a local git repo on a device that supports the MP4 audio encoding format.

git graph --color /dev/null; clear; say "You should never copy and paste terminal commands without proper sanitization. Shame. Shame."; clear;
git graph --with-audio --audio-format mp4
